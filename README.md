# AeroFocus ✈️ — Pomodoro de Vuelo

**AeroFocus** es una herramienta de productividad basada en la técnica Pomodoro, diseñada con una estética de aviación profesional. No es solo un temporizador; es una experiencia de vuelo donde cada sesión de enfoque es un trayecto hacia un destino global.

> **Ignacio Ezequiel Sanchez Edition** — Optimizando el flujo de trabajo, un despegue a la vez.

---

## 🚀 Características Principales

* **Planificación de Vuelo Personalizada:** Configura tiempos de enfoque (Focus), descansos y número de escalas.
* **Mapa de Ruta Dinámico:** Visualiza tu progreso a través de un mapa interactivo basado en coordenadas reales.
* **Conexión con API Real:** Integración con `restcountries.com` para cargar ciudades y capitales de todo el mundo.
* **Sistema de Millas y Rangos:** Acumula millas por cada sesión completada y asciende desde *Cadete* hasta *Comandante*.
* **Modo Noche (Night Mode):** Interfaz adaptativa para sesiones de trabajo nocturnas, protegiendo la fatiga visual.
* **Multiplataforma:** Disponible en versión Web y versión de Escritorio (.exe).

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (Vanilla ES6+).
* **Framework CSS:** [Tailwind CSS](https://tailwindcss.com/) para un diseño responsivo y moderno.
* **API:** REST Countries API para la base de datos de locaciones globales.
* **Desktop:** Empaquetado nativo para Windows.

## 📦 Instalación y Uso

### Versión Web
Puedes acceder directamente a la aplicación en vivo a través de GitHub Pages:
👉 **(https://nachoesnz.github.io/AeroFocus/)**

### Versión de Escritorio (Windows)
1. Ve a la sección de [Releases](https://github.com/Nachoesnz/AeroFocus/releases).
2. Descarga el archivo `aerofocus_0.1.0_x64-setup.exe`.
3. Ejecuta el instalador. 
   * *Nota: Al ser un desarrollador independiente, es posible que Windows SmartScreen muestre una advertencia. Selecciona "Más información" y "Ejecutar de todas formas".*
